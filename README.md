🦭 Seal World

Landing page temática e ilustrativa sobre focas, criada com o objetivo de apresentar, de forma leve e divertida, curiosidades sobre esses mamíferos aquáticos.

📋 Sobre o projeto

Seal World é uma página única (single page) que combina texto, imagens e vídeo de fundo para contar um pouco sobre o mundo das focas: onde vivem, como se movem, suas características físicas e curiosidades sobre a espécie. O projeto tem um tom descontraído, com trocadilhos e uma narrativa amigável.

✨ Seções do site


Hero: vídeo de fundo em loop com o título de destaque "Get Ready to Seal-abrate!"
Exploring the World of Seals: introdução sobre a vida das focas
Style and Moves: sobre a elegância e a camada de gordura (blubber) que as ajuda a se adaptar a diferentes temperaturas
Seal Hangouts: onde encontrar focas — de zonas polares a praias ensolaradas
Meet the Seal Crew: galeria com as principais espécies (foca-comum, foca-cinzenta, elefante-marinho e leopardo-marinho)
Cool Facts About Seals: curiosidades sobre a capacidade de mergulho e o uso dos bigodes como "GPS" subaquático


🛠️ Tecnologias utilizadas


HTML5 — estrutura da página
CSS3 — estilização, layout e responsividade
Google Fonts — fonte Oxygen
Vídeo e imagens em formato nativo (<video> e <img>)

🚀 Como executar o projeto


1. Clone este repositório: git clone https://github.com/GustavoHSS1/Seal-World.git
2. Acesse a pasta do projeto: cd Seal-World
3. Abra o arquivo index.html no seu navegador de preferência.

Não é necessário nenhuma instalação de dependências — o projeto utiliza apenas HTML e CSS puros.
